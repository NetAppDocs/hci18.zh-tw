= 開始部署
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
在繼續NetApp HCI 進行不完整部署之前、您必須先閱讀並瞭解終端使用者授權合約。

.步驟
. 在 * 歡迎使用 NetApp HCI * 頁面上、選取 * 開始使用 * 。
. 在「*先決條件*」頁面上、執行下列動作：
+
.. 確保符合每項先決條件、並選取每個相關核取方塊以確認。
.. 選擇*繼續*。


. 在「*終端使用者授權*」頁面上、執行下列動作：
+
.. 閱讀 NetApp 終端使用者授權合約
.. 如果您接受條款、請在協議文字底部選取 * 我接受 * 。
.. 閱讀VMware終端使用者授權合約。
.. 如果您接受條款、請在協議文字底部選取 * 我接受 * 。
.. 選擇*繼續*。
